## 💡 What is this?

Most language apps teach you to translate English into German word by word. That is not how Germans think.

**Denk auf Deutsch** is different. Every flashcard shows you an English sentence and challenges you to think about how a native German speaker would actually express that idea — including the grammar structure, word order, and cultural nuance behind it.

Each card reveals:
- ✅ The natural German sentence
- ✅ Audio pronunciation in both English and German
- ✅ A grammar note explaining **why** the German is structured that way

---

## ✨ Features

- 🎴 **Flashcard system** — See English, think in German, reveal the answer
- 🔁 **Flip mode** — Switch between English → German and German → English
- 🔊 **Audio for both languages** — Native pronunciation using the Web Speech API
- 📊 **4 proficiency levels** — A1, A2, B1, B2
- 💡 **Grammar notes** — Every card explains the German thinking pattern behind the sentence
- 🌐 **Multi-language ready** — Architecture built to support French, Spanish and more in v2
- 📱 **Fully responsive** — Works on desktop and mobile
- ⚡ **Fast and lightweight** — No database, no login required, no ads

## 🎯 Proficiency Levels

| Level | Description | Example |
|---|---|---|
| **A1** | Complete Beginner | "Where is the toilet?" → "Wo ist die Toilette?" |
| **A2** | Elementary | "I have lived here for two years." → "Ich wohne seit zwei Jahren hier." |
| **B1** | Intermediate | "If I had more time, I would travel more." → "Wenn ich mehr Zeit hätte, würde ich mehr reisen." |
| **B2** | Upper Intermediate | Complex passive voice, Konjunktiv I, formal register |
